# 🚀 Guia de sobrevivéncia: Dominío do Terminal (CLI)
Este repósitorio foi criado como parte da díciplina de **Sistemas Operacionais**, utilizando o simulador [Terminal_Master] (https://andersonrmgomes.github.io/terminal-master/) para patricar comandos essenciais do Windons e LInux.

---

## 📖Introdução 
A interface de linha de comando (CLI) é uma feramenta poderosa que permite interagir com o núcleo do sistema operacional. Dominar esses comandos é fundamental para automação de tarefas, administração de servidores e desenvolvimento de sowftware

---

## 🖥️Comandos Aprendidos

| Comando | Função | Sistema | Exmplo de uso|
| :--- | :--- | :--- | :--- |
| `pwd` | `Exibe o diretorio atual(caminho)` | `Linux` | `pwd` |
| `mkdir` | `Cria diretorios` | `Ambos` | `mldir tolls` |
| `cd` | `Altera o diretório de trabalho` | `Ambos` | `cd/home/usuario/documento` |
| `ls -la` | `Lista tudo` | `Linux` | `ls` |
| `dir` | `Lista tudo` | `Windows` | `dir` |
| `cat` | `Exibe o diretorio atual(caminho)` | `Linux` | `cat readme_hack.md` |
| `type` | `Le o arquivo` | `Windows` | `type readme_hack.txt` |
| `grep` | `Buscar` | `Linux` | `grep "ERRO" logs.txt` |
| `findstr` | `Buscar` | `Windows` | `findstr "ERRO" logs.txt` |
| `chmod` | `Permissões` | `Linux` | `chmod +x script.sh` |
| `icacls` | `Permissões` | `Windows` | `icacls "C:\PastaExemplo` |
| `ps aux` | `Processos` | `Linux` | `ps aux \ grep nome_do_processo` |
| `tasklist` | `Processos` | `Windows` | `tasklist /fi "imagename eq chrome.exe` |
| `find` | `Buscar arquivos` | `Linux` | `find . -name "filename.txt"` |
| `dir /s` | `Busca recursos` | `Windows` | `dir /s` |
| `pipe` | `Encardear` | `Ambos` | `type file.txt \sort` |
| `exit` | `Volta ao menu` | `Ambos` | `exit` |
| `restart` | `recarega missão` | `Ambos` | `restart` |
| `help` | `Exibe ajuda sobre comandos` | `Windows` | `help copy` |
| `ping` | `Teste de conexão com o host` | `Ambos` | `ping google.com` |
| `move` | `Move ou renomeia arquivos/pastas` | `Windows` | `move info.txt dados.txt` |
| `df` | `Exibe uso de espaço em disco ` | `Linux` | `df -h` |
| `top` | `Monitora recussos em tempo real` | `Linux` | `top` |
| `csl` | `Limpa a tela do terminal ` | `Windows` | `cls` |

---

## 🪙Dica de Ouro:
CD: Change Directory, sua utilidade é navegar entre as pastas (diretórios) do seu computador sem precisar usar o mouse ou o gerenciador de arquivos.
